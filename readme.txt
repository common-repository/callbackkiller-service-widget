CallbackKILLER – набор сервисов для увеличения конверсии

== Description ==
CallbackKILLER – это набор сервисов, которые существенно повышают отдачу от Вашего сайта. Используя сервисы, Вы легко получите дополнительные продажи и прибыль без дополнительных вложений в рекламу. 
Включает в себя следующие сервисы:

- Обратный звонок с сайта за 24 секунды (дополнительные звонки с сайта) – прочитать <a href="http://cb-killer.ru/callback/?utm_source=marketplace&utm_medium=wordpress&utm_campaign=callback">подробно</a>;

- Онлайн чат для сайта (переписка с посетителями сайта в реальном времени) прочитать <a href="http://cb-killer.ru/chat/?utm_source=marketplace&utm_medium=wordpress&utm_campaign=chat">подробно</a>;

- Генератор клиентов (дополнительные заявки с сайта) прочитать <a href="http://cb-killer.ru/generator/?utm_source=marketplace&utm_medium=wordpress&utm_campaign=generator">подробно</a>;

- Бесплатный виджет Захватчик (Захватывает внимание посетителя и перенаправляет его на нужные элементы сайта) прочитать <a href="http://cb-killer.ru/invader/?utm_source=marketplace&utm_medium=wordpress&utm_campaign=invader">подробно</a>.

Все виджеты предназначены для повышения отдачи от Вашего сайта. Они гарантировано повысят эффективность работы Вашего сайта.
Все виджеты можно попробовать бесплатно. Подключитесь на сайте – <a href="http://cb-killer.ru/?utm_source=marketplace&utm_medium=wordpress&utm_campaign=connect">Подключиться</a>


== Installation ==
Через консоль:

1. Зайдите "Плагины" > Добавить новый
2. Найдите CallbackKiller
3. Активируйте плагин.

Через FTP:
1. Распакуйте архив с плагином
2. Загрузите папку с плагином в директорию /wp-content/plugins/ на вашем сервере
3. Активируйте плагин CallbackKiller в разделе Плагины в панели администрирования WordPress
